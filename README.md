# 三択コンパイラー

## 実行方法

1. 以下の、urlを開きます。

https://embed.plnkr.co/plunk/2nOl0AYjrVAbLJsK

2.組み合わせを、クリックして、runボタンを押します。

3.生成されたソースコードをコピーします。

4.エディタを開き、ペーストして、*test.py*と実行環境へセーブします。

5.ファイル *test.py* の、`client`のアクセストークンを追記します。

6. *test.py* を実行する

```shell
$ python test.py
```
